# Iconițe PWA (placeholder)

Aici trebuie adăugate iconițele reale, referite din `vite.config.ts` (manifest):

- `icon-192.png` — 192×192
- `icon-512.png` — 512×512
- `icon-512-maskable.png` — 512×512, `purpose: maskable`

Sunt necesare pentru instalarea PWA și pentru magazine (App Store / Play Store).
Până la design-ul final, pot fi generate rapid din `favicon.svg`.
